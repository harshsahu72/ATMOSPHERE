🌤️ ATMOSPHERE — Weather App
A sleek, dark-themed weather web app built with pure HTML No frameworks, no build tools — just open the file and go

✨ Features
🔍 Smart Search — Search any city, state, country, or region with live autocomplete suggestions
🌡️ Current Weather — Temperature, feels like, weather description, high/low for the day
💨 Wind & Conditions — Wind speed & direction, humidity, precipitation, visibility, UV index
🫁 Air Quality Index (AQI) — European AQI with pollutant breakdown (PM2.5, PM10, NO₂, O₃)
⏱️ Hourly Forecast — Next 24 hours with weather icons and precipitation probability
📅 7-Day Forecast — Daily high/low temperatures and weather conditions
🗺️ Interactive Map — Leaflet.js map with switchable layers: Temperature, Wind, Clouds, Precipitation, and AQI overlay
🕐 Search History — Recently searched locations saved as quick-access chips
📱 Responsive Design — Works on desktop and mobile
